# Creating an OTA

## Overview
1. Create the boot img file (kernel, ramdisk, command line arguments)
2. Create the sytem fs file
3. Create a manifest
4. Create the tarball
5. Test the barball


## Creating the ramdisk image

Unzipping an existing RAM disk to a folder:

    gzip -dc ../boot.img-ramdisk.gz | cpio -imd

Re-zipping it up

    find . ! -name . | LC_ALL=C sort | cpio -o -H newc -R root:root | gzip > ../new-boot.img-ramdisk.gz


## Creating bootimg file

    mkbootimg --kernel ./1.7-boot.img-zImage \
    --ramdisk ./1.7-dev-boot.img-ramdisk.gz \
    --cmdline "$(cat ./rcm-cmdline)" \
    --base "81c00000" \
    --pagesize "2048" \
    --ramdisk_offset "00008000" \
    --second_offset "00f00000" \
    --tags_offset "00000100" \
    --output ./1.7-dev-boot.img
